# RAG 自動化系統 - 快速使用指南

## 🚀 5分鐘快速開始

### 步驟 1: 獲取應用程序
1. 找到文件: `RAG_Automation_System_20260124_071605.zip`
2. 大小: 約 0.3 MB
3. 位置: `auto-rag-system/dist/` 目錄

### 步驟 2: 部署到 Windows 電腦
1. 將 ZIP 文件複製到 Windows 電腦
2. 右鍵點擊 ZIP 文件 → "解壓縮到當前文件夾"
3. 建議解壓到: `C:\Programs\RAG_System\`

### 步驟 3: 安裝應用程序
1. 進入解壓後的目錄
2. 雙擊運行 `install_rag_app.bat`
3. 按照提示完成安裝
4. 安裝程序會自動:
   - 檢查 Python 環境
   - 創建桌面快捷方式
   - 創建開始菜單快捷方式

### 步驟 4: 啟動應用程序
1. 雙擊桌面上的「RAG 自動化系統」快捷方式
2. 或從開始菜單啟動
3. 或運行目錄中的 `start.bat`

## 🎯 第一次使用

### 1. 選擇測試項目
1. 啟動應用程序
2. 點擊「瀏覽...」按鈕
3. 選擇目錄: `auto-rag-system/test_project/`
4. 點擊「確定」

### 2. 運行基礎分析
1. 點擊「🚀 基礎分析」按鈕
2. 確認分析對話框
3. 等待分析完成 (約 1-2 分鐘)
4. 查看輸出日誌中的進度

### 3. 查看結果
1. 分析完成後查看桌面
2. 找到文件: `RAG_系統結果_時間戳.txt`
3. 打開文件查看分析摘要
4. 詳細報告在 `output/` 目錄中

## 🆘 緊急問題解決

### 應用程序無法啟動
1. **檢查 Python**:
   ```cmd
   python --version
   ```
   應該顯示 Python 3.8+

2. **重新安裝**:
   - 刪除當前安裝
   - 重新運行 `install_rag_app.bat`
   - 以管理員身份運行

3. **手動啟動**:
   ```cmd
   cd "C:\Programs\RAG_System"
   python rag_app.py
   ```

### 分析過程卡住
1. **等待**: 大型項目可能需要較長時間
2. **檢查日誌**: 查看輸出日誌中的錯誤信息
3. **重啟**: 關閉應用重新啟動
4. **縮小範圍**: 分析項目子目錄

### 報告未生成
1. **檢查桌面權限**: 確保有寫入桌面權限
2. **查看輸出目錄**: 檢查 `output/` 目錄
3. **檢查日誌**: 查看錯誤信息
4. **重新分析**: 再次運行分析
