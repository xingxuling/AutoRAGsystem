=======================================
    RAG系统 - Windows桌面快捷方式使用说明
=======================================

✅ 桌面快捷方式已成功创建!

桌面上现在有两个快捷方式:

1. RAG-System.lnk
   - 快速启动版
   - 双击直接启动RAG系统
   - 简单易用

2. RAG-System-Full.lnk
   - 完整菜单版
   - 双击进入功能菜单
   - 包含所有高级功能

=======================================
    使用方法
=======================================

方法1: 使用桌面快捷方式(推荐)
-----------------------------
1. 在桌面上找到 RAG-System.lnk 或 RAG-System-Full.lnk
2. 双击快捷方式
3. 系统会自动启动并检测Python环境
4. 根据提示操作即可

方法2: 直接运行批处理文件
-----------------------------
1. 打开文件夹: C:\Users\User\auto-rag-system
2. 双击以下任一文件:
   - 启动RAG系统.bat (快速启动)
   - RAG_Windows_Launcher.bat (完整菜单)

=======================================
    系统要求
=======================================

- Python 3.8 或更高版本
- Windows 7/8/10/11

如未安装Python:
1. 访问 https://www.python.org/downloads/
2. 下载并安装最新版本
3. 安装时务必勾选 "Add Python to PATH"

=======================================
    功能说明
=======================================

RAG系统主要功能:

1. 项目分析
   - 分析项目结构
   - 评估代码质量
   - 检查功能完整性

2. 智能决策
   - 优先级评估
   - 迭代建议
   - 风险评估

3. 自动优化
   - 项目结构优化
   - 配置文件更新
   - 文档改进

4. 自动打包
   - 生成优化版本
   - 输出到桌面
   - 生成详细报告

=======================================
    输出文件位置
=======================================

分析结果将保存在:
- 桌面: 优化后的项目包和报告
- C:\Users\User\auto-rag-system\output\: 详细分析结果
- C:\Users\User\auto-rag-system\logs\: 执行日志

=======================================
    故障排除
=======================================

问题1: 双击快捷方式没有反应
解决: 右键快捷方式 -> 属性 -> 检查目标路径是否正确

问题2: 提示Python未找到
解决: 安装Python 3.8+ 并确保勾选 "Add Python to PATH"

问题3: 中文显示乱码
解决: 系统已设置UTF-8编码,如仍有问题请检查系统区域设置

=======================================
    技术支持
=======================================

如遇到问题:
1. 查看日志文件: C:\Users\User\auto-rag-system\logs\
2. 检查输出目录: C:\Users\User\auto-rag-system\output\
3. 运行诊断脚本: C:\Users\User\auto-rag-system\diagnose_install.bat
