# 项目截图

请在此目录添加项目截图，用于展示在 README 中。

## 推荐截图内容

1. **主界面** - 展示系统的主要功能界面
2. **分析报告** - 展示项目分析结果
3. **优化结果** - 展示优化前后的对比
4. **使用演示** - 展示如何使用系统

## 截图要求

- 格式：PNG 或 JPG
- 尺寸：建议 800x400 或 1200x600
- 命名：使用描述性名称，如 `main-interface.png`

## 上传后更新 README

上传截图后，请在 README.md 中更新截图链接：

```markdown
![主界面](./screenshots/main-interface.png)
```
