## 🚀 安裝步驟

### 步驟1: 複製到Windows電腦
1. 將ZIP文件從以下位置複製到您的Windows電腦：
   ```
   C:\Users\User\auto-rag-system\dist\Enhanced_RAG_System_Windows_20260124_042407.zip
   ```

### 步驟2: 解壓縮
1. 在Windows電腦上，右鍵點擊ZIP文件
2. 選擇「解壓縮到當前文件夾」或「解壓縮到...」
3. 建議解壓縮到：`C:\Programs\Enhanced_RAG_System\`

### 步驟3: 安裝應用
1. 進入解壓縮後的目錄
2. 雙擊運行 `install.bat`
3. 按照提示完成安裝
4. 安裝程序會：
   - 檢查Python環境
   - 創建桌面快捷方式
   - 創建開始菜單快捷方式

### 步驟4: 啟動應用
1. 雙擊桌面上的「增強版RAG系統」快捷方式
2. 或者從開始菜單啟動

## 🔧 配置自定義

### 修改配置文件
編輯 `config.json`：

```json
{
  "system": {
    "name": "Enhanced RAG System",
    "version": "1.0.0",
    "windows": true
  },
  "defaults": {
    "project_dir": "C:\\Users\\YourName\\projects",
    "output_dir": "./output"
  }
}
```

### 自定義監測設置
創建 `monitoring_config.json`：

```json
{
  "watch_directories": [
    "C:\\Users\\YourName\\projects",
    "C:\\Users\\YourName\\workspace"
  ],
  "exclude_patterns": [
    "node_modules",
    ".git",
    "__pycache__",
    ".venv"
  ],
  "interval_seconds": 5
}
```

## ⚡ 性能優化

### 減少資源占用
1. 增加監測間隔：
   ```json
   "interval_seconds": 10
   ```

## 🔍 故障排除

### 常見問題解決方案

#### 問題1: Python未找到
```
錯誤：未找到Python
```
**解決**：
1. 安裝Python 3.8+
2. 安裝時勾選「Add Python to PATH」
3. 重新啟動電腦
4. 重新運行安裝程序

#### 問題2: 權限不足
```
無法創建桌面快捷方式
```
**解決**：
1. 以管理員身份運行 `install.bat`
2. 或手動創建快捷方式

#### 問題3: 分析失敗
```
分析過程中出現錯誤
```
**解決**：
1. 檢查項目路徑是否正確
2. 確保對項目目錄有讀取權限
3. 查看錯誤日誌進行調試

#### 問題4: 中文顯示亂碼
```
中文顯示為亂碼
```
**解決**：
1. 確保Windows系統語言設置正確
2. 使用支持UTF-8的終端
3. 修改批處理文件的編碼設置
